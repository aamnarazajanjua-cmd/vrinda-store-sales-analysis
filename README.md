# 🌸 Vrinda Store Sales Analysis Dashboard

##  Overview

This project analyzes sales data from **Vrinda Store** using Microsoft Excel. The dashboard provides interactive visualizations and business insights to help understand customer purchasing behavior, sales performance, and order trends.



## Dataset

- Sales Orders
- Customer Details
- Product Categories
- Order Status
- Sales Channels
- State-wise Sales



##  Objectives

- Clean and prepare raw sales data
- Analyze monthly sales trends
- Understand customer demographics
- Compare sales channels
- Identify top-performing states
- Create an interactive Excel dashboard



##  Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting
- Excel Formulas



## Dashboard

The dashboard includes:

-  Monthly Sales Trend
-  Sales by Gender
-  Sales by Age Group
-  Top States by Sales
-  Order Status Analysis
-  Sales by Channel

- Women customers contribute the highest sales.

- Amazon is the best-performing sales channel.
- Maharashtra generates the highest revenue.
- Most customer orders are successfully delivered.
